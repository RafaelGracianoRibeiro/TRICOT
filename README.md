# Instituto de Psicologia Tricot - Landing Page

![Logo do Instituto Tricot](./images/Logo-marca.png)

## Sobre o Projeto

Este é o projeto da landing page para o Instituto de Psicologia Tricot. A página foi desenvolvida para apresentar o instituto, seus profissionais e sua missão, utilizando a metáfora do "tricô" para representar o cuidado e a reconstrução do bem-estar emocional.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

-   `index.html`: Arquivo principal que estrutura toda a landing page.
-   `README.md`: Este arquivo, com a documentação do projeto.
-   `Símbolo.png`: Símbolo da marca.
-   `styles/`: Pasta contendo todas as folhas de estilo CSS.
    -   `global.css`: Estilos aplicados globalmente em toda a página.
    -   `header.css`: Estilos específicos para o cabeçalho.
    -   `hero.css`: Estilização da seção de destaque principal (hero section).
    -   `profissionais.css`: Estilos para a seção que apresenta os profissionais.
    -   `footer.css`: Estilos para o rodapé da página.
-   `images/`: Pasta para armazenar as imagens utilizadas no site, como logos e fotos.
-   `icons/`: Pasta para os ícones utilizados na interface.

## Tecnologias Utilizadas

-   HTML5
-   CSS3
